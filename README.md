# Student Task Manager

## Problem Statement
Students often forget assignments, projects, and daily study tasks. This project helps students manage their tasks in one place.

## Target User
- College Students
- Polytechnic Students
- School Students

## Features (MVP)
- Add New Task
- View All Tasks
- Edit Task
- Delete Task
- Store Data in MySQL
- Responsive Design
- PHP Server-side Validation

## Technologies Used
- HTML5
- CSS3
- JavaScript
- PHP
- MySQL
- XAMPP

## How to Run

1. Install XAMPP.
2. Start Apache and MySQL.
3. Open phpMyAdmin.
4. Import database.sql.
5. Copy the project folder to htdocs.
6. Open:

http://localhost/Student-Task-Manager/
